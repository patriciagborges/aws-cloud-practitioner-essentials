Diferentes tipos de instancias do EC2: Cada tipo de instancia do Amazon EC2 é agrupado em uma família de instancias e são otimizadas para determinados tipos de tarefas. Os tipos de instancias oferecem combinações variadas de CPU, memoria, armazenamento e capacidade de rede. Famílias de instancias do Amazon EC2: Uso geral; Otimizada para computação; Otimizada para memória; Computação acelerada; Otimizada para armazenamento. Uso geral: Recursos equilibrados; Cargas de trabalho diversas; Servidores web; Repositórios de códigos. Servidores de aplicações; Servidores de back-end para aplicações corporativos. Banco de dados pequenos e médios. Otimizada para computação: Tarefas de uso intenso de computação; Servidores de jogos; Oferece processadores de alta performance; Computação de Alta performance (HPC); Modelagem científica. Otimizada para memória: Ideal para bancos de dados de alta performance Uso intenso de memória. Computação acelerada: Calculos de números de ponto flutuante; Processamento de gráficos; Correspondências de padrões de dados; Utilizar aceleradores de hardware. Otimizada para armazenamento: Adequada para aplicações de data warehousing; Alta performance para dados armazenados localmente.
